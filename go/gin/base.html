2. 特性

特性 说明
快速 基于 Radix 树的路由(httprouter)，小内存占用。没有反射。可预测的 API 性能。
支持中间件 传入的 HTTP 请求可以由一系列中间件和最终操作来处理。 例如：Logger，Authorization，GZIP，最终操作 DB。
Crash 处理 Gin 可以 catch 一个发生在 HTTP 请求中的 panic 并 recover 它。这样，你的服务器将始终可用。例如，你可以向 Sentry 报告这个 panic！
JSON 验证 Gin 可以解析并验证请求的 JSON，例如检查所需值的存在。
路由组 更好地组织路由。是否需要授权，不同的 API 版本…… 此外，这些组可以无限制地嵌套而不会降低性能。
错误管理 Gin 提供了一种方便的方法来收集 HTTP 请求期间发生的所有错误。最终，中间件可以将它们写入日志文件，数据库并通过网络发送。
内置渲染 Gin 为 JSON，XML 和 HTML 渲染提供了易于使用的 API。
可扩展性 新建一个中间件非常简单。

3. 使用 jsoniter 编译

Gin 使用 encoding/json 作为默认的 json 包，但是你可以在编译中使用标签将其修改为 jsoniter。

go build -tags=jsoniter .

4. 路由

4.1. 简单路由

func main() {
    r := gin.Default()
    r.GET("/get", func(c *gin.Context) {
        c.JSON(http.StatusOK, gin.H{
            "message": "success",
        })
    })

    r.Run(":8888")
}

4.2. 路由组

func main() {
    r := gin.Default()
    v1 := r.Group("v1")
    // /v1/get
    v1.GET("/get", func(c *gin.Context) {
        c.JSON(http.StatusOK, gin.H{
            "message": "success",
        })
    })

    r.Run(":8888")
}

5. 参数解析

5.1. GET请求

例如：/get?username=Lsc&age=18。[Query/DefaultQuery/QueryMap/QueryArray/BindQuery]

type Info struct {
    Name string `form:"name"`
    Age  int    `form:"age"`
    Arr  []int  `form:"arr"`
}

func main() {
    r := gin.Default()
    r.GET("/get", func(c *gin.Context) {
        // 获取GET参数
        name := c.Query("name")
        //if name, ok := c.GetQuery("name"); !ok {
        //    panic("name不存在")
        //}

        // 获取GET参数，设置默认值
        age := c.DefaultQuery("age", "18")
        // 获取GET参数，map格式：other[a]=1234&other[b]=hello
        other := c.QueryMap("other")
        // 获取GET参数，数组格式：arr=1&arr=2&arr=3
        arr := c.QueryArray("arr")
        // 绑定到结构体
        var info = Info{}
        if err := c.BindQuery(&info); err != nil {
            panic(err)
        }

        c.JSON(http.StatusOK, gin.H{
            "name":  name,
            "age":   age,
            "other": other,
            "arr":   arr,
            "info":  info,
        })
    })

    r.Run(":8888")
}

5.2. Post请求

5.2.1. Form表单提交

multipart/form-data 当需要上传文件时，使用该编码。数据以指定的分隔符分开，数据也无需编码转换，它采用了键值对的方式。 CURL请求如下：

curl --location --request POST 'http://127.0.0.1:8888/post' \
--form 'name="Lsc"' \
--form 'age="18"' \
--form 'other[first]="22"' \
--form 'other[two]="hello"' \
--form 'arr="1"' \
--form 'arr="2"'

HTTP请求格式如下：

POST /post HTTP/1.1
Host: 127.0.0.1:8888
Content-Length: 567
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW

----WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="name"

Lsc
----WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="age"

18
----WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="other[first]"

22
----WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="other[two]"

hello
----WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="arr"

1
----WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="arr"

2
----WebKitFormBoundary7MA4YWxkTrZu0gW

application/x-www-from-urlencoded 默认情况下，使用该编码，将表单内的数据转换为键值对，&分隔。 CURL请求如下：

curl --location --request POST 'http://127.0.0.1:8888/post' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--data-urlencode 'name=Lsc' \
--data-urlencode 'age=18' \
--data-urlencode 'other[first]=22' \
--data-urlencode 'other[two]=hello' \
--data-urlencode 'arr=1' \
--data-urlencode 'arr=2'

HTTP请求格式如下：

POST /post HTTP/1.1
Host: 127.0.0.1:8888
Content-Type: application/x-www-form-urlencoded
Content-Length: 68

name=Lsc&age=18&other%5Bfirst%5D=22&other%5Btwo%5D=hello&arr=1&arr=2

Go代码如下：

type Other struct {
    First int    `form:"other[first]"`
    Two   string `form:"other[two]"`
}

type PostInfo struct {
    Name  string `form:"name"`
    Age   int    `form:"age"`
    Arr   []int  `form:"arr"`
    Other Other
}

func main() {
    r := gin.Default()
    r.POST("/post", func(c *gin.Context) {
        // 接收Post参数
        name := c.PostForm("name")
        // 接收Post参数，设置默认值
        age := c.DefaultPostForm("age", "18")
        // 接收Post参数，接收map：
        other := c.PostFormMap("other")
        // 接收Post参数，接收array：
        arr := c.PostFormArray("arr")
        // 接收Post参数，绑定到结构体，
        var p = PostInfo{}
        if err := c.ShouldBind(&p); err != nil {
            panic(err)
        }

        c.JSON(http.StatusOK, gin.H{
            "name":  name,
            "age":   age,
            "other": other,
            "arr":   arr,
            "p":     p,
        })
    })

    r.Run(":8888")
}

5.2.2. application/json

这种请求方式是现在实际项目中常用的。 CURL请求如下：

curl --location --request POST 'http://127.0.0.1:8888/post' \
--header 'Content-Type: application/json' \
--data-raw '{
    "name":"lsc",
    "age":12,
    "other":{
        "first":12,
        "two":"hello"
    },
    "arr":[1,2,3]
}'

HTTP请求如下：

POST /post HTTP/1.1
Host: 127.0.0.1:8888
Content-Type: application/json
Content-Length: 116

{
    "name":"lsc",
    "age":12,
    "other":{
        "first":12,
        "two":"hello"
    },
    "arr":[1,2,3]
}

Go代码如下：


type JsonOther struct {
    First int    `form:"first"`
    Two   string `form:"two"`
}

type JsonPostInfo struct {
    Name      string    `form:"name"`
    Age       int       `form:"age"`
    Arr       []int     `form:"arr"`
    JsonOther JsonOther `form:"other" json:"other"`
}

func main() {
    r := gin.Default()
    r.POST("/post", func(c *gin.Context) {
        // 从c.Request.Body读取请求数据
        b, err := c.GetRawData()
        if err != nil {
            panic(err)
        }
        fmt.Println(string(b))
        // 定义map或结构体
        var m JsonPostInfo
        // 反序列化
        err = json.Unmarshal(b, &m)
        if err != nil {
            panic(err)
        }

        c.Request.Body = ioutil.NopCloser(bytes.NewBuffer(b))
        // 接收Post参数，绑定到结构体，
        var p = JsonPostInfo{}
        //var p = PostInfo{}
        if err := c.ShouldBind(&p); err != nil {
            panic(err)
        }

        c.JSON(http.StatusOK, gin.H{
            "m": m,
            "p": p,
        })
    })

    r.Run(":8888")
}

5.2.3. 获取Path参数

请求如下：

http://127.0.0.1:8888/get/8/lsc

Go代码如下:

func main() {
    r := gin.Default()
    r.GET("/get/:id/:name", func(c *gin.Context) {
        name := c.Param("name")
        id := c.Param("id")

        c.JSON(http.StatusOK, gin.H{
            "name": name,
            "id":   id,
        })
    })

    r.Run(":8888")
}
